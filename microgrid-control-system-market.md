<p><p>Microgrid control systems are classified into three main types based on their connection to the main power grid. Grid-connected microgrids are connected to the main grid and can both import and export power. Off-grid microgrids operate independently of the main grid, relying solely on local energy sources. Hybrid microgrids combine aspects of both grid-connected and off-grid systems, enabling them to utilize the main grid as a backup while also being able to function independently. Each type offers unique advantages and capabilities for different applications.</p></p>
